# 격자에서 주유소를 통과하는 최단경로의 수 찾기

## 문제 설명

격자 모양의 거리가 있다.

편의상 각 교차점의 위치를 평면 좌표 `(x, y)` 로 나타낸다.  
교차점 중 두 곳엔 지정된 주유소 `A`, `B`가 있다.

출발점 `S(sx, sy)` 와 도착점 `T(tx, ty)` 가 주어질 때,  
`S` 에서 `T` 로 가는 최단경로 중에서 다음 조건을 만족하는 경로의 개수를 구하라.

- 주유소 `A` 또는 `B` 중 적어도 하나를 지나야 한다.
- 주유소가 최단경로 상에 존재하지 않으면 해당 주유소는 통과할 수 없다.

---

## 입력

첫 번째 줄에 테스트 케이스의 수 `T` 가 주어진다.

각 테스트 케이스는 다음과 같이 주어진다.

```text
sx sy tx ty ax ay bx by
```

### 제한 조건

- `T ≤ 10000`
- `1 ≤ sx, sy, tx, ty ≤ 1000`
- `1 ≤ ax, ay, bx, by ≤ 1000`

---

## 출력

각 테스트 케이스마다 다음 값을 출력한다.

- 출발점에서 도착점까지 가는 서로 다른 최단경로의 수
- 단, 주유소 `A` 또는 `B` 를 적어도 하나 지나야 함
- 결과는 `1000007` 로 나눈 나머지를 출력

---

## 입력 예시

```text
4 5 6 9 5 5 8 9

0 0 1000 1000 10 9 9 10

```

# 풀이 아이디어

## 최단경로 개수 구하기

격자에서 `(x1, y1)` 에서 `(x2, y2)` 로 가는 최단경로는:

- 오른쪽 이동 `dx`
- 위쪽 이동 `dy`

를 섞는 경우의 수와 같다.

즉,

```math
\binom{dx+dy}{dx}
```

이다.

---

## 핵심 아이디어

우리가 원하는 것은:

```text
A를 지나거나 B를 지나는 최단경로
```

이다.

포함 배제 원리를 사용하면:

```text
(A를 지나는 경로)
+ (B를 지나는 경로)
- (A와 B를 모두 지나는 경로)
```

가 된다.

---

## 최단경로 판별

점 `P` 가 `S → T` 최단경로 위에 존재하려면:

```text
dist(S,P) + dist(P,T) == dist(S,T)
```

이어야 한다.

여기서 거리 함수는 맨해튼 거리이다.

```text
dist(A,B)=|x1-x2|+|y1-y2|
```

# 사용 알고리즘

## 조합(DP)

조합값을 미리 계산한다.

```text
C[n][r]
```

을 파스칼 삼각형 방식으로 구한다.

---

# Python 코드

```python
import sys

MOD = 1000007
MAX = 2000

# 조합 미리 계산
C = [[0] * (MAX + 1) for _ in range(MAX + 1)]

for i in range(MAX + 1):
    C[i][0] = 1
    C[i][i] = 1

    for j in range(1, i):
        C[i][j] = (C[i - 1][j - 1] + C[i - 1][j]) % MOD


def dist(p, q):
    return abs(p[0] - q[0]) + abs(p[1] - q[1])


def ways(p, q):
    dx = abs(p[0] - q[0])
    dy = abs(p[1] - q[1])

    return C[dx + dy][dx]


def path_count(points):
    total = dist(points[0], points[-1])

    dsum = 0
    result = 1

    for i in range(len(points) - 1):
        dsum += dist(points[i], points[i + 1])

        result = (
            result * ways(points[i], points[i + 1])
        ) % MOD

    # 전체 경로가 최단경로인지 확인
    if dsum == total:
        return result

    return 0


T = int(sys.stdin.readline())

for _ in range(T):
    sx, sy, tx, ty, ax, ay, bx, by = map(
        int,
        sys.stdin.readline().split()
    )

    S = (sx, sy)
    T_point = (tx, ty)

    A = (ax, ay)
    B = (bx, by)

    count_A = path_count([S, A, T_point])
    count_B = path_count([S, B, T_point])

    count_AB = path_count([S, A, B, T_point])
    count_BA = path_count([S, B, A, T_point])

    answer = (
        count_A
        + count_B
        - count_AB
        - count_BA
    ) % MOD

    print(answer)
```

---

# 시간 복잡도

## 조합 전처리

```text
O(MAX^2)
```

- `MAX = 2000`

---

## 각 테스트 케이스

거리 계산 및 조합 조회만 수행하므로:

```text
O(1)
```

---

## 전체 시간 복잡도

```text
O(MAX^2 + T)
```

충분히 빠르게 동작한다.
