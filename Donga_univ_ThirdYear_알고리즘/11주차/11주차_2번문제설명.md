# 양수를 제곱의 합으로 나타내기

## 문제 설명

모든 양수는 서로 다른 양수의 제곱의 합으로 나타낼 수 있다.  
예를 들어, `5 = 2^2 + 1^2` 로 나타낼 수 있다.

어떤 양의 정수 `n` 이 주어질 때 이를 양수의 제곱의 합으로 나타내는 방법이 1가지 이상 있을 수 있다.

예를 들어, `11`은 아래와 같이 세 가지 서로 다른 식으로 표현이 가능하다.

1. `11 = 1^2 + 1^2 + ... + 1^2`
2. `11 = 2^2 + 2^2 + 1^2 + 1^2 + 1^2`
3. `11 = 3^2 + 1^2 + 1^2`

각 표현에서 더해지는 항(term)의 개수는 각각 `11`, `5`, `3`개이다.

임의의 정수 `n`에 대하여 다음 조건을 만족하는 표현을 출력하는 프로그램을 작성하라.

- 더해지는 항의 수가 최소여야 한다.
- 같은 항 개수의 표현이 여러 개라면,
  - `b1`이 가장 큰 표현을 선택한다.
  - `b1`도 같다면 `b2`가 큰 표현,
  - 그 다음은 `b3` ...
- 즉, 사전순 내림차순으로 가장 큰 표현을 선택한다.

---

## 입력

입력의 각 줄에 서로 다른 양수 `n`이 주어진다.

- `n ≤ 100000`
- 입력값이 `0`이면 입력 종료

---

## 출력

각 입력값 `n`에 대해 아래 형식으로 출력한다.

```text
n=b1^2+b2^2+...+bk^2
```

- 공백은 포함하지 않는다.
- 항의 개수는 최소여야 한다.
- 같은 항 개수라면 사전순으로 가장 큰 표현을 선택한다.

---

## 입력 예시

```text
10
28
1113
0
```

---

## 출력 예시

```text
10=3^2+1^2
28=5^2+1^2+1^2+1^2
1113=32^2+8^2+5^2
```

# 풀이 아이디어

## 핵심 조건

우선순위는 다음과 같다.

1. 항 개수 최소
2. 첫 번째 수 최대
3. 두 번째 수 최대
4. ...

즉, 가능한 표현 중에서:

- 길이가 가장 짧고
- 사전순 내림차순으로 가장 큰 수열

을 찾아야 한다.

---

## 사용 알고리즘

### DFS(백트래킹)

라그랑주 네 제곱수 정리에 의해 모든 수는 최대 4개의 제곱수 합으로 표현 가능하다.

따라서:

- 1개 제곱수
- 2개 제곱수
- 3개 제곱수
- 4개 제곱수

순서로 탐색하면 처음 찾는 답이 곧 최적해가 된다.

또한 큰 수부터 탐색하면 사전순 조건도 자동으로 만족한다.

---

# Python 코드

```python
import math

def solve(n):
    limit = math.isqrt(n)

    for k in range(1, 5):
        result = dfs(n, limit, k)

        if result:
            return result


def dfs(remain, max_root, count):
    if count == 0:
        if remain == 0:
            return []
        return None

    for x in range(min(max_root, math.isqrt(remain)), 0, -1):
        rest = dfs(remain - x * x, x, count - 1)

        if rest is not None:
            return [x] + rest

    return None


while True:
    n = int(input())

    if n == 0:
        break

    result = solve(n)

    expression = "+".join(str(x) + "^2" for x in result)

    print(str(n) + "=" + expression)
```

---

# 시간 복잡도

최대 4개의 제곱수만 탐색하므로 실제 탐색 깊이는 매우 작다.

- 최악의 경우: `O(sqrt(n)^4)`
- 하지만 가지치기와 깊이 제한 때문에 실제로는 매우 빠르게 동작한다.

`n ≤ 100000` 범위에서 충분히 통과 가능하다.
