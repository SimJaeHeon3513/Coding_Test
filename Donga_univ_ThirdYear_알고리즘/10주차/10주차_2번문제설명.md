# 직각다각형

## 문제 설명

N개의 꼭지점으로 이루어진 다각형에서 각 꼭지점의 내각이 `90°` 또는 `270°`이며,  
어느 두 변도 서로 교차하지 않고, 각 꼭지점의 좌표가 서로 다른 다각형을 **직각다각형**이라 부른다.

예를 들어, 아래 그림 (a)와 같은 8개의 꼭지점으로 이루어진 직각다각형을 고려해보자.

각 변에 적혀 있는 수는 그 변의 길이를 나타낸다.

직각다각형을 표현하는 한 가지 방법은 다음과 같다.

- 임의의 꼭지점에서 시작
- 임의의 방향으로 변을 따라 이동
- 각 변의 길이와,
- 그 변에서 다음 변으로 이동할 때의 회전 방향 정보

를 차례로 기록한다.

회전 방향은 다음과 같이 표기한다.

- `L` : 좌회전 (Left turn)
- `R` : 우회전 (Right turn)

---

## 예시 1

그림 (b)처럼 붉은 화살표의 시작 꼭지점에서 시작하여  
화살표 방향으로 다각형의 변을 따라 이동하면 다음과 같은 열을 얻는다.

```txt
(2, R), (3, L), (2, L), (2, R), (3, R), (4, R), (7, R), (5, R)
```

---

## 예시 2

그림 (c)처럼 시작 꼭지점과 방향이 다르면 다음과 같은 열을 얻는다.

```txt
(4, L), (3, L), (2, R), (2, R), (3, L), (2, L), (5, L), (7, L)
```

---

물론 이 두 열은 동일한 직각다각형을 나타내는 서로 다른 표현이다.

변의 길이를 `l`이라고 할 때,

- `(l, L)` 은 양수 `+l`
- `(l, R)` 은 음수 `-l`

으로 나타내면 직각다각형 정보는 N개의 정수로 구성된 리스트로 표현할 수 있다.

위 예시의 두 표현은 각각 다음과 같다.

```txt
(-2, 3, 2, -2, -3, -4, -7, -5)
(4, 3, -2, -2, 3, 2, 5, 7)
```

---

각각 N개의 정수로 구성된 두 정수 리스트 `S1`, `S2`가 주어질 때,

- 두 리스트가 동일한 직각다각형을 표현하는지
- 아니면 서로 다른 직각다각형을 표현하는지

판별하는 프로그램을 작성하라.

---

## 입력

입력의 첫 번째 줄에는 테스트케이스의 수를 나타내는 정수 `T`가 주어진다.

```txt
1 ≤ T ≤ 20
```

각 테스트케이스의 첫째 줄에는 다각형의 꼭지점 개수를 나타내는 정수 `N`이 주어진다.

```txt
4 ≤ N ≤ 20,000
```

이후 두 줄에 걸쳐 길이 `N`의 정수 리스트가 주어진다.

정수 리스트의 각 원소는 문제에서 설명한 방식으로 표현된 값이며,

- 절댓값은 `1 이상 1000 이하`
- 각 리스트는 항상 올바른 직각다각형을 표현

함이 보장된다.

---

## 출력

각 테스트케이스마다 한 줄에 하나의 정수를 출력한다.

- 두 리스트가 동일한 직각다각형을 표현하면 `1`
- 서로 다른 직각다각형을 표현하면 `0`

을 출력한다.

---

## 예시 입력

```txt
2
8
-2 3 2 -2 -3 -4 -7 -5
4 3 -2 -2 3 2 5 7
8
-2 3 2 -2 -3 -4 -7 -5
-2 3 2 -2 -3 -4 -5 -7
```

---

## 예시 출력

```txt
1
0
```

---

## 설명

첫 번째 테스트케이스의 두 리스트는 시작 위치와 진행 방향만 다를 뿐,  
동일한 직각다각형을 표현한다.

두 번째 테스트케이스는 서로 다른 다각형을 나타낸다.
